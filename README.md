# Расписание

СТАРЫЙ КОД!

#### Автоматизированное расписание занятий
#### C#
#### Платформа MAUI
#### База данных: SQLite

#### Функционал:
1. Удобный, минималистичный интерфейс.
2. Автоматическое статус дня недели и даты.
3. Автоматическое открытие вкладки дня занятий(т.е. ненужно кликать и искать какие занятия).
4. Автоматическое определение времени занятий(по раннее введенному шаблону).
5. Ввод и вывод информации о предмете, преподавателе, аудитории.
6. Статус-строка пары(в зависимости от времени показывает, состояние пар: начало пары, перемену, конец пары).
7. Возможность введения шаблона расписания занятий(раскрывается автоматический функционал приложения).

#### Порядок работы:
1. Заходим во вкладку расписание звонков
   - Вводим время для начала и конца пары в формате 00:00
   - Перезагружаем приложения для корректной работы
2. Во вкладке Расписание занятий выбираем любой день недели и по кнопке +
   - Вводим занятие (первым знаком должно быть "число: 1, 2, 3 и т.д." - для определения пары)
   - Вводим преподавателя
   - Вводим аудиторию
3. Удаление занятия происходит по выбору занятия и кнопке -
